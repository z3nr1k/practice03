# IT-Cube Миасс — Репозиторий проектов обучающихся

## О нас
Центр цифрового образования «IT-куб» — структурное подразделение МАУ ДО «ДДТ «Юность» им. В.П. Макеева» (г. Миасс, пр. Макеева, 39)[reference:9]. Реализуется в рамках федерального проекта «Цифровая образовательная среда» национального проекта «Образование»[reference:10][reference:11].

## Направления обучения
- Программирование на Python[reference:12]
- Программирование на Java[reference:13]
- Мобильная разработка[reference:14]
- Виртуальная и дополненная реальность (VR/AR)[reference:15]
- Системное администрирование[reference:16]
- Цифровая гигиена и работа с большими данными[reference:17]
- Базовые навыки программирования на C-подобных языках[reference:18]
- Робототехника[reference:19]

## Интеллектуальные партнёры
Проекты разрабатываются при поддержке партнёров: Яндекс, Samsung, Крибрум, Фирма 1С, Кванториумы (Челябинск, Магнитогорск), компания «Папилон» и другие IT-компании[reference:20][reference:21].
